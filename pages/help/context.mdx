
# Understanding context

Context object can be used to get reference to your databases,collections and values.
Tip: you can use keywords like rdb,rcoll,rval to automatically write these syntax.

To get reference to your databases
```js 
const user_db = context.services.get().db("<your DB name>")
```

To get your collection reference
```js
const user_collection = context.services.get().db("<your DB name>").collection("<Your collection name>")

```

To get collection to your environment values and secret

```js
const secret_key = context.values.get("ValueName")

```

To call some other pre defined function from current function

```js
const context.functions.execute("Function Name",...function params)

#example
const userId= "someId"
const userPosts =await context.functions.execute("getUserPost",userId)
const userProfileData = await context.functions.execute("getUserProfileData",userId)
```
