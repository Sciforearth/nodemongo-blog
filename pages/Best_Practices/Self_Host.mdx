# Self host

nodemongo's self host is a part of no vendor lock in mechanism.

You can download your entire backend from your app's landing page and directly deploy it at your preferred hosting provider.

### why self host?

1. It's free.
2. It's way faster >>
3. It's fully compatible with our SDK's
4. It's more reliable.
5. It's customizable.


### What can you do with Self Host?
1. Security is configured for functions
2. You can create endpoints with the template provided 
3. Endpoint security are also configured
4. All types of authentication covered
5. Easy to use functions to reduce your development time

### what are the limitaions?
Self host has some minor limitaions, one of which is authentication and all functionalities related to authentication is still managed by nodemongo for security purposes
