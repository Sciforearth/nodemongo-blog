# Introduction
 
Welcome to nodemongo! This is the documentation for getting you started with nodemongo with simple documentation.
## What is Nodemongo?

A **simple**, **powerful** , **fast**  and **secure** way to create your backend in an instant. Using Nodemongo you spend 70% less time creating your backend which helps you focus on adding more features and make your product amazing  !
